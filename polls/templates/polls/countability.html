Countability

Introduction

It is important to set up the background information necessary to begin understanding countability. When we say two sets are of the same size, we are saying they have the same cardinality. Sets can either hold a finite or infinite number of items, therefore, this cardinality may also be infinite or finite. Two sets can be related together by a function which can take an item from one set and return an item from the other depending on the regulations of this function.

Assume we have a function, f, for two sets, C and D such that:

f : C D
1-1, Injective
  • every element in the codomain is mapped to by at most one element in the domain
  • no item in D is mapped to by two items in C
  • |C| |D|
Onto, Surjective
  • every element in the codomain is mapped to by at least one element in the domain
  • every item in D is mapped to by at least one item in C
  • |C| |D|
f : C D
Bijective
  • both injective and surjective
  • every item in D is mapped to by exactly 1 item in C and vice versa
  • |C| |D|

The Pigeonhole Principle is when you have two sets, S and T, and a function, f, as follows:

f : S T

If |S| |T|, than f cannot be 1-1. This is because there are too many pigeons but not enough pigeonholes. There must be at least one hole (in T) shared by 2 pigeons (from S).

What Does It Mean For A Set To Be Countable?

A set is countable if:

  • It is finite
  • It has a bijection with the natural numbers (that means it is countably infinite)
  • It is a subset of a countable set

Why Is It Important To Know If A Set Is Countable?

For anything to be computable by Java (or any other Turing-complete computing language) the set of possibilities must be countable. This means that the Java program must have a countable number of acceptable values in order to decide whether or not an inputted value is accepted.

Therefore, for us to code and use deciders within computer science, we have to be able to determine whether the acceptable items would be countable. Now, let’s look into how we determine whether or not a set is countable.

Methods For Determining Countability

If a set is known to be finite, we can conclude that it is a countable set. When a set is infinite, however, this must be proven a bit further. It is possible for infinite sets to be countable, like the natural numbers, ℕ, for example, but it is also possible for infinite sets to be uncountable, like all real numbers, ℝ.

Through finding a bijection, we are able to prove that there does exist a bijective function that maps the given set to the natural numbers. Through diagonalization, we are able to prove that there exists no possible bijection between the given set and the natural numbers.

TL;DR:

  • proving an infinite set is countable find a bijection with ℕ
  • proving an infinite set is not countable diagonalization
