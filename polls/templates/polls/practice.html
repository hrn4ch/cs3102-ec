Can You Count Em?

  • 1. Are the even numbers countable?

    • If S is the set of even natural numbers, then:

    f : ℕ S

    f(x) = 2x

  • 2. Are the odd numbers countable?

    • If S is the set of odd natural numbers, then:

    f : ℕ S

    f(x) = 2x-1

  • 5. Is the set of all languages countable?

    • This can be proven using diagonalization. Assume towards reaching a contradiction that I have listed all languages. I can always generate a language that I have not listed by doing the opposite for the ith string in the language L(i). That way no language listed is equivalent, and I have reached a contradiction. Therefore, through diagonalization, the set of all languages is not countable.
